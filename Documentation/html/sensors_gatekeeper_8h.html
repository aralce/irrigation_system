◆ sensorsGatekeeper_getName()

bool sensorsGatekeeper_getName ( uint8_t  ID)

‪Obtiene el nombre del Sensor.

Esta función se utiliza para ordenar al sensorsGatekeeper que entregue el nombre del sensor deseado. El nombre del sensor será enviado al Datalogger.

Parámetros
ID‪Identificador del sensor. Para los sensores físicos debe estar comprendido entre 0 y 6. Siendo 0 el "Sensor 1" y 6 el "Sensor 7"
Valores devueltos
True‪Devuelve true si la orden se envía con normalidad al sensorsGatekeeper.
False‪Devuelve false si hubo un problema. Probablemente el ID no sea válido.
